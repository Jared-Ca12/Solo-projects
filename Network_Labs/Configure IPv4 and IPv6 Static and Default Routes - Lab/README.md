# 15.6.2 Lab - Configure IPv4 and IPv6 Static and Default Routes
## This Lab was done in person using actual hardware instead of Packet Tracer

## Files in this folder
- JCaldwell_15.6.2 Lab - Configure IPv4 and IPv6 Static and Default Routes.docx

## Objectives
Part 1: Build the Network and Configure Basic Device Settings

Part 2: Configure and verify IP and IPv6 addressing on R1 and R2

Part 3: Configure and verify static and default routing for IPv4 on R1 and R2

Part 4: Configure and verify static and default routing for IPv6 on R1 and R2

## Summary of Steps Taken
In this lab I cabled all the devices. Then, I did basic configurations on the switches and routers. Then, I enabled routing and assigned the correct IP address to all the interfaces. Then, I configured IPv4 static and default routes on R1 and R2, and verified the routes. Finally, I did the same for IPv6.
