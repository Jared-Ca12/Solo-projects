# 11.6.2 Lab - Switch Security Configuration
## This Lab was done in person using actual hardware instead of Packet Tracer

## Files in this folder
- JCaldwell_11.6.2 Lab - Switch Security Configuration.docx

## Objectives
Part 1: Configure the Network Devices.

Cable the network.

Configure R1.

Configure and verify basic switch settings.

Part 2: Configure VLANs on Switches.

Configure VLAN 10.

Configure the SVI for VLAN 10.

Configure VLAN 333 with the name Native on S1 and S2.

Configure VLAN 999 with the name ParkingLot on S1 and S2.

Part 3: Configure Switch Security.

Implement 802.1Q trunking.

Configure access ports.

Secure and disable unused switchports.

Document and implement port security features.

Implement DHCP snooping security.

Implement PortFast and BPDU guard.

Verify end-to-end-connectivity.

## Summary of Steps Taken
In this lab, I cabled all the devices. Then, I configured the router and switches with basic configurations. Then, I set up the VLANs for both switches and assigned the correct ports to them. Then, I configured the trunking ports. Then, I configured port security settings on the two ports that connect to end devices. Then, I enabled dhcp snooping on S2. Then, I enabled portfast and bpduguard on the two ports that connect to end devices. Finally, I verified connectivity.
