## Objectives
Part 1: Create a Secure Trunk

Part 2: Secure Unused Switchports

Part 3: Implement Port Security

Part 4: Enable DHCP Snooping

Part 5: Configure Rapid PVST PortFast and BPDU Guard

## Summary of Steps Taken
In this lab, first I configured my trunk port and added them to the same native vlan. Then, I shutdown the unused ports and added them to the correct vlan. Then, I enabled port security on the access ports and set the maximum number of mac addresses. Finally, I enable portfast and bpdu guard.
