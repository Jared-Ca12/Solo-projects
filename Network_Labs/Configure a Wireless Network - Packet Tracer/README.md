- JCaldwell_13.5.1 Packet Tracer - Configure a Wireless Network.docx

## Objectives
Connect to a wireless router

Configure the wireless router

Connect a wired device to the wireless router

Connect a wireless device to the wireless router

Add an AP to the network to extend wireless coverage

Update default router settings

## Summary of Steps Taken
In this lab, I configured basic settings for a wireless router. Then, I configured the wireless settings and the wireless security settings. Then, I configured an access point and connected clients to the network. Finally, I changed the dhcp pool information and reconnected the devices.
