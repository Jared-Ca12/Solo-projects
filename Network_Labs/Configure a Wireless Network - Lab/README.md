# 13.1.11 Lab - Configure a Wireless Network
## This Lab was done in person using actual hardware 

## Files in this folder
- JCaldwell_13.1.11 Lab - Configure a Wireless Network.docx

## Objectives
In this lab, you will configure basic settings on a wireless router and connect a PC to the router wirelessly, including configuring the SSID, wireless security, DHCP settings, and adding an access point to extend wireless coverage.

## Summary of Steps Taken
In this lab, first I cabled the devices. Then, I configured the router information, the ip address, and the dhcp settings. Then, I verified connectivity and turned one router into an access point. Finally, I disabled the broadcast.
