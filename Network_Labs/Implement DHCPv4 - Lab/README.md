# 7.4.2 Lab - Implement DHCPv4
## This Lab was done in person using actual hardware instead of Packet Tracer

## Files in this folder
- JCaldwell_7.4.2 Lab - Implement DHCPv4.docx

## Objectives
Part 1: Build the Network and Configure Basic Device Settings

Part 2: Configure and verify two DHCPv4 Servers on R1

Part 3: Configure and verify a DHCP Relay on R2

## Summary of Steps Taken
In this lab, first I created an address scheme and cabled all the devices. Then, I did the basic configurations on the switches and routers. Then, I configured the interfaces, VLANs, and trunk ports on the switches and routers. Then, I set up a DHCP server on the network. Finally, I verified connectivity.
